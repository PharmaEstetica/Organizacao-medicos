# CORREÇÃO: SUPORTE A IMAGENS HEIC (iPhone)

## Problema

- Arquivos `.heic` / `.heif` (formato padrão do iPhone) não exibem preview no navegador
- Navegadores só suportam nativamente: JPG, PNG, GIF, WEBP

---

## Solução

Converter HEIC para JPEG no momento do upload usando a biblioteca `heic2any`.

---

## Implementação Frontend

### 1. Instalar dependência
```bash
npm install heic2any
```

### 2. Atualizar componente de Upload
```tsx
// components/ImageUpload.tsx

import { useState, useRef } from 'react';
import { Upload, X, User, Loader2 } from 'lucide-react';
import heic2any from 'heic2any';

interface ImageUploadProps {
  currentImage?: string;
  onImageChange: (file: File | null) => void;
}

export function ImageUpload({ currentImage, onImageChange }: ImageUploadProps) {
  const [preview, setPreview] = useState<string | null>(currentImage || null);
  const [converting, setConverting] = useState(false);
  const inputRef = useRef<HTMLInputElement>(null);

  const handleFileChange = async (e: React.ChangeEvent<HTMLInputElement>) => {
    const file = e.target.files?.[0];
    if (!file) return;

    // Validar tamanho (10MB para HEIC pois são maiores)
    if (file.size > 10 * 1024 * 1024) {
      alert('Imagem muito grande. Máximo 10MB.');
      return;
    }

    let processedFile: File = file;

    // Se for HEIC/HEIF, converter para JPEG
    if (
      file.type === 'image/heic' || 
      file.type === 'image/heif' ||
      file.name.toLowerCase().endsWith('.heic') ||
      file.name.toLowerCase().endsWith('.heif')
    ) {
      try {
        setConverting(true);
        
        const convertedBlob = await heic2any({
          blob: file,
          toType: 'image/jpeg',
          quality: 0.85
        });

        // heic2any pode retornar array ou blob único
        const blob = Array.isArray(convertedBlob) ? convertedBlob[0] : convertedBlob;
        
        // Criar novo File com extensão .jpg
        const newFileName = file.name.replace(/\.heic$/i, '.jpg').replace(/\.heif$/i, '.jpg');
        processedFile = new File([blob], newFileName, { type: 'image/jpeg' });
        
        setConverting(false);
      } catch (error) {
        setConverting(false);
        console.error('Erro ao converter HEIC:', error);
        alert('Erro ao processar imagem do iPhone. Tente converter para JPG antes de enviar.');
        return;
      }
    }

    // Validar tipo final
    const allowedTypes = ['image/jpeg', 'image/jpg', 'image/png', 'image/webp'];
    if (!allowedTypes.includes(processedFile.type)) {
      alert('Formato não suportado. Use JPG, PNG ou WEBP.');
      return;
    }

    // Criar preview
    const reader = new FileReader();
    reader.onloadend = () => {
      setPreview(reader.result as string);
    };
    reader.readAsDataURL(processedFile);

    onImageChange(processedFile);
  };

  const handleRemove = () => {
    setPreview(null);
    onImageChange(null);
    if (inputRef.current) {
      inputRef.current.value = '';
    }
  };

  return (
    <div className="flex flex-col items-center mb-6">
      <div
        className="relative w-28 h-28 rounded-lg overflow-hidden border-2 border-dashed border-gray-300 cursor-pointer hover:border-green-500 transition-colors"
        onClick={() => !converting && inputRef.current?.click()}
      >
        {converting ? (
          <div className="w-full h-full flex flex-col items-center justify-center bg-gray-100">
            <Loader2 className="animate-spin text-green-500" size={32} />
            <span className="text-xs text-gray-500 mt-2">Convertendo...</span>
          </div>
        ) : preview ? (
          <>
            <img
              src={preview}
              alt="Preview"
              className="w-full h-full object-cover"
            />
            <button
              type="button"
              onClick={(e) => {
                e.stopPropagation();
                handleRemove();
              }}
              className="absolute top-1 right-1 bg-red-500 text-white rounded-full p-1 hover:bg-red-600"
            >
              <X size={16} />
            </button>
          </>
        ) : (
          <div className="w-full h-full flex flex-col items-center justify-center bg-gray-100 text-gray-400">
            <Upload size={32} />
            <span className="text-xs mt-1 text-center px-2">
              Clique ou arraste
            </span>
          </div>
        )}
      </div>

      <input
        ref={inputRef}
        type="file"
        accept="image/jpeg,image/jpg,image/png,image/webp,image/heic,image/heif,.heic,.heif"
        onChange={handleFileChange}
        className="hidden"
        disabled={converting}
      />

      <p className="text-xs text-gray-500 mt-2 text-center">
        JPG, PNG, WEBP ou HEIC (iPhone)<br />
        Máx. 10MB
      </p>
    </div>
  );
}
```

---

## Formatos Aceitos

| Formato | Suporte | Ação |
|---------|---------|------|
| `.jpg` / `.jpeg` | ✅ Nativo | Usa direto |
| `.png` | ✅ Nativo | Usa direto |
| `.webp` | ✅ Nativo | Usa direto |
| `.heic` | ⚠️ iPhone | Converte para JPG |
| `.heif` | ⚠️ iPhone | Converte para JPG |

---

## Fluxo de Conversão
```
1. Usuário seleciona arquivo .HEIC do iPhone
              ↓
2. Sistema detecta formato HEIC
              ↓
3. Mostra "Convertendo..." com spinner
              ↓
4. heic2any converte para JPEG (qualidade 85%)
              ↓
5. Preview exibe imagem convertida
              ↓
6. Upload envia arquivo JPEG (não HEIC)
```

---

## Observações Importantes

- A conversão acontece **no navegador** (client-side)
- O arquivo enviado ao servidor será sempre **JPG** (já convertido)
- Isso evita problemas no backend que não suporta HEIC
- A qualidade 85% mantém boa definição com tamanho menor
